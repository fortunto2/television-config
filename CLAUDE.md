# television-config

Custom TV channels repo. Config lives at `~/.config/television/`.

## Structure
- `config.toml` — main tv config (keybindings, UI, shell integration triggers)
- `cable/claude-*.toml` — custom channels (tracked in git)
- `cable/*.toml` — standard channels from `tv update-channels` (gitignored)
- `scripts/` — helper scripts, installed to `~/.local/bin/`

## Custom Channels
- `claude-sessions` — recent sessions with preview + resume (from .jsonl transcripts)
- `claude-history` — full prompt history, 6+ months (from history.jsonl)
- `claude-memory` — search CLAUDE.md, auto-memory, rules (via rg, cached 5min)

## Adding a Channel
1. Create `cable/my-channel.toml` with `[metadata]`, `[source]`, `[preview]`, `[actions]`
2. If needs a helper script — add to `scripts/` and `~/.local/bin/`
3. Optional: add shell trigger in `config.toml` under `[shell_integration.channel_triggers]`
4. `git add && git commit && git push`

## Conventions
- Channel names: kebab-case
- Script names: `tv-{channel-name}` prefix
- Display paths: shorten `$HOME` prefixes for readability
- Standard channels are regenerated by `tv update-channels` — never edit them
